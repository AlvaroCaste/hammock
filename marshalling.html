Marshalling

Hammock has the notion of marshalling at several levels. First, it provides the Codec typeclass, for low level operations, and the MarshallF algebra for unmarshalling operations suspended in Free.

Codecs

For marshalling operations, Hammock provides Encoder and Decoder typeclasses:

@typeclass trait Encoder[A] {
  def encode(a: A): Entity
}

@typeclass trait Decoder[A] {
  def decode(a: Entity): Either[CodecException, A]
}

These typeclasses convert values to and from Hammock Entity.

Circe integration

Hammock is integrated with circe, you can use this integration with the hammock-circe package. This integration is designed to be non-intrusive, since a simple import of import hammock.circe.implicits._ will give you all the funcionality you need. For example:

scala> import hammock.{Encoder => HammockEncoder, Decoder => HammockDecoder, _}
import hammock.{Encoder=>HammockEncoder, Decoder=>HammockDecoder, _}

scala> import hammock.circe.implicits._
import hammock.circe.implicits._

scala> import io.circe.{Encoder => CirceEncoder, Decoder => CirceDecoder}
import io.circe.{Encoder=>CirceEncoder, Decoder=>CirceDecoder}

scala> case class Car(brand: String, model: String)
defined class Car

scala> implicit val carEncoder: CirceEncoder[Car] = CirceEncoder.forProduct2("brand", "model") { car =>
     |   (car.brand, car.model)
     | }
carEncoder: io.circe.Encoder[Car] = io.circe.ProductEncoders$$anon$2@1bf8c664

scala> implicit val carDecoder: CirceDecoder[Car] = CirceDecoder.forProduct2("brand", "model")(Car)
carDecoder: io.circe.Decoder[Car] = io.circe.ProductDecoders$$anon$2@5abe00e9

scala> val ent: Entity = HammockEncoder[Car].encode(Car("ford", "fiesta"))
ent: hammock.Entity = StringEntity({"brand":"ford","model":"fiesta"},hammock.ContentType$$anon$2@4747b02c)

scala> val Right(fordFiesta) = HammockDecoder[Car].decode(ent)
fordFiesta: Car = Car(ford,fiesta)

Marshalling algebra

Hammock also provides a marshalling algebra. This algebra only provides an operation, unmarshall that transforms Entityes into As for which we have a Decoder.
